🚀 Quick Start Guide

  1. Upload: Select your PDF file using the button below.
  2. Extract: Enter page numbers (or leave blank for all) and click "Extract Text".
  3. Format: Click "Smart Format with AI" to automatically structure the extracted content.
  4. Refine & Export: Edit, summarize, copy, or download your formatted text.

Advanced: Extract Specific Text

In the "Extracted Text" box, right-click the first word of a section, then right-click the last word to select a range. The selected text will appear in the "Formatted Text" box.
